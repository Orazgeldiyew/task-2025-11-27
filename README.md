# Link Status Checker

Небольшой сервис на Go для проверки доступности интернет-ссылок.  
Принимает наборы URL, проверяет их статус и сохраняет результаты на диск.  
Для каждого набора присваивается ID, по которому можно сформировать PDF-отчёт.

Работает без базы данных, состояние хранится в `data/state.json`.  
Незавершённые задачи обрабатываются после перезапуска.

## Возможности

- приём одного или нескольких URL за один запрос  
- назначение уникального ID каждому набору  
- асинхронная проверка ссылок через worker-pool  
- хранение состояния между перезагрузками  
- генерация PDF-отчётов по списку ID  
- Swagger-документация

## Запуск

```bash
go run ./cmd/server
