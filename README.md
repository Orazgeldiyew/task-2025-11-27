Веб-сервис принимает от пользователя одну или несколько ссылок, проверяет их доступность, сохраняет историю запросов и позволяет выгружать PDF-отчёты по ранее созданным наборам ссылок.

Сервис работает асинхронно: запросы складываются в очередь, обрабатываются worker-пулами и сохраняются на диск, благодаря чему данные не теряются при перезагрузке сервиса.
